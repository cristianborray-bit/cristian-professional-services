## Deploy
### Opción 1: Vercel (Recomendado)
```bash
npm i -g vercel
vercel --prod
```

### Opción 2: Netlify
```bash
npm i -g netlify-cli
netlify deploy --prod --dir .
```

### Opción 3: GitHub Pages
1. Crear repo `cristian-professional-services`
2. Push código
3. Activar GitHub Pages en settings
